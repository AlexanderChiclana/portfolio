Project Description

A freelance project for Catalyst Miami, a nonprofit organization developing a campaign for clean energy in Miami-Dade. I was responsible for overhauling the 1.0 version of the site which had been generated using a rudimentary website creation tool specifically for non-profits. While developing this site, I needed to learn how to apply my own code to an extensive computer-generated codebase in a way that wouldn’t affect the sites functionality.

Project Details

  • Integrated Bootstrap 4 components
  • Created interactive panels that change image and trigger modals on click
  • Overrode default styling to improve responsiveness and UX
  • Restructured repository and hosted on amazon s3

Hi I’m Alex, I’m a full stack developer with a background in human factors research. Since graduating from Brown University in 2016 I independently studied interface design through the lens of user psychology. As my fascination with user experience design grew, I found myself falling deeper into the world of programming. During the summer of 2018 I was accepted into General Assembly’s Web Development Immersive fellowship, a 500+ hour program where I learned how to create full stack web apps from scratch. Some of the technologies I’ve worked with include: HTML5, CSS3, JavaScript ES6, Express.js, Ruby, and Ruby on Rails among others. While my strengths are in front end development, I have as solid grasp on back end fundamentals and am looking to expand my skills further. Currently I am learning React.js and freelancing.
